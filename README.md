# India Climate Data Analysis (1901–2020)

## Overview
This project analyzes long-term climate trends in India using IMD datasets. It covers annual and seasonal temperature changes from 1901–2020, visualized in Tableau dashboards.

## Data
- **Source**: India Meteorological Department (IMD)
- **File**: `TEMP_ANNUAL_Raw_Data.csv`
- **Columns**: Year, Annual Mean Temperature, Seasonal Means (Jan–Feb, Mar–May, Jun–Sep, Oct–Dec)

## Visualizations
- **Annual Trend Line**: Shows gradual warming trend post-1970s.
- **Seasonal Comparison**: Highlights variability across monsoon and winter seasons.
- **Heatmap of Anomalies**: Detects seasonal deviations and recent winter declines.
- **Decadal Box Plot**: Demonstrates rising medians and fewer low outliers in recent decades.

## Insights
- Median temperatures rise sharply after the 1970s.
- 2000s and 2010s decades show consistently higher averages.
- Winter anomalies decline after the 2000s, while monsoon variability persists.

## How to Reproduce
1. Clone this repository.
2. Open `climate data analysis in india - 1900 to 2020.twb` in Tableau.
3. Connect to `data/TEMP_ANNUAL_Raw_Data.csv`.
4. Explore dashboards and filters.
